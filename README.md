# Client Registry Frontend

This is the frontend repository for the Client Registry project. It is implemented in React with TypeScript and Tailwind CSS.

## Deployment

Client Registry Frontend is deployed on an Digital Ocean Ubuntu server with Nginx. You can access it at [http://sandeep-sharma.me](http://sandeep-sharma.me).

## Packages Used

- ```React Query```: Used for managing server state and API responses.
- ```Zod``` for schema validation, ensuring that data sent to the backend adheres to the expected structure and format.
- ```React Hook Form``` for easy management of form state and validation.
- ```shadcn/ui``` for UI building.
